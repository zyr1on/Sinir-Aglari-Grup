# Snake Reinforcement Learning Project

This repository contains implementations of reinforcement learning agents playing the classic Snake game. The project is organized into two main folders, each with a different focus: multi-agent experiments and DQN variant comparisons.

## Repository Structure

### `multi_agent_racing/`

This folder contains experiments with multiple Snake agents learning in parallel under the same environment. All agents share the same DQN architecture but differ in learning strategies:

- **Fast Learner** – agent with a higher learning rate, adapts quickly to the environment.  
- **Slow Learner** – agent with a lower learning rate, learns more gradually.  
- **Greedy** – agent with a tendency to exploit its current knowledge rather than explore.  
- **Standard** – baseline agent using default DQN hyperparameters.

These experiments allow observation of how different learning speeds and exploration strategies affect performance when agents compete in the same environment.

### `different_dqn_semih/`

This folder focuses on comparing different DQN-based algorithms with the same Snake environment:

- **DQN** – the standard Deep Q-Network.  
- **DDQN** – Double DQN to reduce overestimation bias.  
- **Rainbow DQN** – a combination of multiple DQN improvements for better performance.

The goal of this folder is to evaluate and compare the performance of various DQN variants while keeping the environment and architecture consistent.

## Usage

1. Install the required dependencies listed in `requirements.txt`.  
2. Run the desired experiment by executing the Python script in the corresponding folder.  ( `train.py`)
3. Monitor training progress and performance metrics through the logs or plots generated by the scripts.  

## Notes

- All agents use the same environment and input representation.  
- Hyperparameters are defined in each folder to differentiate learning strategies or algorithm variants.  
- This repository can serve as a starting point for experimentation with reinforcement learning algorithms in grid-based environments.
